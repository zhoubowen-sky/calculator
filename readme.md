## 灵动计算器APP的简要说明

### 开发工具
- Cordova
- Android Studio
- Webpack
- NodeJS

### APP类型
这是一个Hybrid APP，采用JS编写，使用的前端框架为Framework7，使用的打包工具为Cordova/PhoneGap，可以同时编译打包为Android、IOS以及其他平台的应用，由于是使用前端H5、CSS、JS开发，所以能做到一次开发，多平台使用。
